# AI Habit Tracker 🚀

A Full-Stack AI-Powered Habit Tracker built with the MERN Stack (MongoDB, Express.js, React.js, Node.js), Tailwind CSS v4, and Google Gemini AI.

Track habits, build consistency, visualize progress with GitHub-style heatmaps, and receive personalized AI-powered insights to improve your daily routine.

## ✨ Features

### 🔐 Authentication

* Secure User Registration & Login
* JWT Authentication
* Password Hashing with bcrypt
* Profile Management

### 🎯 Habit Management

* Create New Habits
* Edit Existing Habits
* Archive Habits
* Delete Habits
* Custom Categories
* Habit Icons & Colors
* Flexible Frequencies
* Target Day Selection

### ✅ Daily Tracking

* One-click Habit Completion
* Daily Progress Tracking
* Confetti Celebrations
* Progress Rings

### 🔥 Streak System

* Current Streak Tracking
* Longest Streak Records
* Active Streak Counter
* Streak Recovery Detection

### 📊 Analytics & Visualization

* GitHub-style 90-Day Heatmap
* Weekly Habit Grid
* Monthly Statistics
* Category Distribution Charts
* Performance Insights
* Week-over-Week Comparisons

### 🤖 AI Features (Powered by Google Gemini)

* AI Weekly Performance Report
* Personalized Habit Suggestions
* AI Streak Recovery Coach
* Habit Analysis Chat Assistant
* AI Morning Motivation Messages

### 🎨 UI/UX

* Modern Glassmorphism Design
* Aurora Background Effects
* Light & Dark Theme Support
* Fully Responsive Layout
* Mobile Bottom Navigation
* Smooth Animations

## ⚙️ Environment Variables

Create a `.env` file in the server directory.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

GEMINI_API_KEY=your_google_gemini_api_key
```

## 🚀 Installation

### Clone Repository

```bash
git clone <your-repository-url>
cd habit-tracker
```

### Install Backend Dependencies

```bash
cd server
npm install
```

### Install Frontend Dependencies

```bash
cd ../client
npm install
```

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
